# NBA Prop Analysis

Read `../../sports/nba/props-overview.md`, `../../sports/nba/props-workflow.md`, `../../sports/nba/props-role-and-minutes.md`, and `../../sports/nba/props-red-flags.md`.

## Workflow
1. Define the exact player prop, side, line, and price.
2. Build the thesis from role, minutes, matchup, pace, and prop-family context.
3. Surface only the factors that materially affect the selected prop.
4. Keep recent-form narratives subordinate to role and matchup evidence.

## Return
- prop thesis
- key supporting factors
- key NBA prop uncertainties
