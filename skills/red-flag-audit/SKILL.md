# Red Flag Audit

Read `../../core/red-flags.md` and the relevant sport module.

## Workflow
1. Check for stale or missing critical inputs.
2. Check for thesis-to-market mismatch.
3. Check for weak or narrative-driven reasoning with insufficient evidence.
4. Prioritize only the red flags that materially affect the gate.

## Return
- critical red flags
- medium red flags
- likely effect on the verdict
