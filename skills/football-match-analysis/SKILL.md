# Football Match Analysis

Read `../../sports/football/overview.md`, `../../sports/football/coverage-modes.md`, and `../../sports/football/workflow.md`.

## Workflow
1. Define the exact football event and market.
2. Classify the environment as `strong-stat-coverage` or `weak-stat-coverage`.
3. Build the match read from team strength, chance creation, chance concession, tactical fit, availability, and schedule context.
4. Keep xG-led reasoning only where the coverage mode supports it.

## Return
- analysis mode
- football thesis
- key supporting factors
- major football-specific uncertainties
