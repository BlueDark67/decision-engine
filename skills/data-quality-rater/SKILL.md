# Data Quality Rater

Read `../../core/data-quality.md` and the relevant sport docs.

## Workflow
1. Evaluate completeness, freshness, adequacy to market, coverage strength, and context stability.
2. Rate the setup as `strong`, `usable`, `weak`, or `insufficient`.
3. Explain the rating with only the highest-impact reasons.

## Return
- data quality rating
- short explanation
- main gaps still affecting the setup
