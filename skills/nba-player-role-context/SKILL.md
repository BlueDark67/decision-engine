# NBA Player Role Context

Read `../../sports/nba/props-role-and-minutes.md`, `../../sports/nba/restricted-secondary-scope.md`, and `../../sports/nba/workflow.md`.

## Workflow
1. Review injury status and timestamps.
2. Review projected starters, rotation shape, and likely minute range.
3. Identify usage or touch changes caused by absences or returns.
4. Decide whether role stability supports the prop or should block it.

## Return
- role summary
- minutes stability note
- injury and rotation impact note
