# Market Price Evaluator

Read `../../core/edge-quality.md`, `../../core/decision-gate.md`, and the relevant `markets.md` file in `../../sports/`.

## Workflow
1. Match the thesis to the exact market being discussed.
2. Check whether the current price still fits the thesis.
3. Distinguish a valid sports read from a valid betting price.
4. Note whether the number is still playable, gone, or only watchlist-worthy.

## Return
- market fit
- price discipline note
- whether the current number still deserves evaluation
