# Edge Quality Rater

Read `../../core/edge-quality.md`.

## Workflow
1. Count the independent supports behind the edge.
2. Check how sensitive the thesis is to small price changes.
3. Confirm that the market matches the thesis.
4. Rate the edge as `clear`, `marginal`, `unclear`, or `none`.

## Return
- edge quality rating
- short explanation
- note on price sensitivity
