# sports/nba/props-red-flags.md

## Core red flags
- outdated injury report
- unclear minutes restriction
- role volatility
- teammate return that changes touches or usage
- blowout risk
- foul-risk exposure that threatens minutes
- line inflation caused by recent hot streaks
- combo prop hiding a weak component
- made-threes variance being understated
- thesis too dependent on overtime or an extreme game script

## Effect
- lower confidence
- downgrade `enter` to `watchlist`
- downgrade `watchlist` to `pass`
- in severe cases force `insufficient data`

## Rule
One major role or minutes red flag can be enough to block a prop entry even when the matchup looks appealing.
