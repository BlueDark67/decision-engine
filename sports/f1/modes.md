# sports/f1/modes.md

## Pre-Practice
### Core inputs
- track type
- baseline form
- weather outlook
- penalties or news

### Rule
Conservative mode by default.

## Post-Practice
### Core inputs
- practice pace
- long-run pace
- degradation signals
- initial readiness

### Rule
It already allows stronger reads, but they are still conditional.

## Post-Qualifying
### Core inputs
- grid
- confirmed one-lap pace
- updated penalties
- track-position context

### Rule
This is often the cleanest mode for several H2H and placement markets.

## Pre-Race
### Core inputs
- final weather
- final penalties
- strategic context
- reliability and starting conditions

### Rule
Final gate before entry.
